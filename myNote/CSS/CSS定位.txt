一.定位  
  定位:改变元素在页面中的位置
  分类:
    普通流(文档流)定位:默认定位方式
    浮动定位/相对定位/绝对定位/固定定位
  1.普通流定位
    特点:
      每个元素在页面中都是在父元素左上角显示
      每个元素在页面中一定都有自己的空间
      块级元素独占一行,从上往下排列
      行内元素和行内块元素从左往右排列
  2.浮动定位 float
    (1)特点:
      ①浮动元素会脱离文档流,不再占据原有空间,其他未浮动的元素将上前补位
      ②浮动元素会停靠在其父元素的左/右边,或其它已浮动元素的边缘上
      ③浮动定位能解决的问题:让多个块级元素在一行中显示
    (2)float详解
      取值:none 默认值,无浮动
           left 左浮动,让元素停靠在父元素的左边,或挨着左侧已经浮动的元素边缘
	   right 右浮动,让元素停靠在父元素的右边,或挨着右侧已经浮动的元素边缘
        注意:元素的顺序决定浮动的顺序
      特殊效果:
        ①浮动换行,前面元素的高度会影响第二行的浮动效果
	②未指定元素宽度的情况下,元素浮动后的宽度以内容为主
        ③元素一旦浮动起来,将变为块级元素,允许设置尺寸,正常处理垂直外边距
        ④文本,行内元素,行内块元素采用环绕的方式排列,不会被浮动元素覆盖
    (3)清除浮动 clear
      取值:left/right/both 清除当前元素前面元素左/右/所有浮动所带来的影响  
           none 不清除浮动
      浮动元素对父元素高度造成的影响：
        元素的高度，都是以未浮动元素的高度为准,浮动元素是不占页面空间的，一旦子元素浮动起来，那么父元素将会塌陷
	解决方法：
	  ①给父元素设置高度(在已知父元素高度的情况下)
	  ②设置父元素也浮动,但其他后续元素会上前补位
	  ③设置父元素的overflow属性
	    弊端:如果有内容要溢出显示,也会一同隐藏起来
	  ④在父元素中,加一个空的块级元素,并为其设置clear:both
  3.定位 position
    取值:static 静态,默认值
         relative 相对定位
	 absolute 绝对定位
	 fixed 固定定位
    偏移属性  top/bottom/left/bottom
      取值:px
      例:top:150px
      注意:只有已定位元素,才能使用偏移属性
    (1)相对定位 relative
      相对定位:元素相对于它原来的位置偏移某个距离
      使用场合:在做元素位置的微调时使用
      语法:position:relative;
           left:10px;
    (2)绝对定位 absolute
      使用场合:弹出菜单,有堆叠效果的元素
      特点:
        ①绝对定位的元素会脱离文档流,不在占据页面空间
        ②绝对定位的元素会相对于最近已定位的祖先元素去实现位置的初始化,如果没有已定位的祖先元素,那么元素会相对于body去实现位置的初始化
	③一般子元素要实现绝对定位,父元素通常都设置为相对定位
	④绝对定位会将元素变为块级元素
      语法:position:absolute;
           left:100px;
      堆叠顺序 z-index
        元素定位后,可能会形成堆叠的效果,可以使用z-index改变默认的堆叠顺序
	取值:无单位数字,数字越大,越靠上
	注意:只有已定位的元素,才能使用z-index属性;父子元素间,z-index无效,子元素在父元素上;给子元素设置z-index负值时(父元素不设置z-index),可以实现将父元素在上方
    (3)固定定位 fixed
      固定定位:将元素固定在页面(窗口)的某个位置,不会随着滚动条而发生位置的变化
      语法:position:fixed
      特点:固定定位元素会脱离文档流,不再占据页面空间且变成块级元素;固定定位是相对body定位,与父元素无关
